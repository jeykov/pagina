TEMÁTICAS DESTACADAS

CURSO DE DISEÑO WEB

El participante aprenderá el uso correcto de las etiquetas HTML para estructurar una página web y darle formato mediante hojas de estilos CSS. Esto con la finalidad de contar con un sitio web navegable, atractivo y dinámico.

CURSO DE PROGRAMACIÓN

En este curso de programación aprenderás a emplear el lenguaje de programación Python, para aplicar las habilidades fundamentales de la programación. Es fácil de aprender y solo se requiere tener conocimientos elementales.

CURSO DE WINDOWS 11

Aprende a usar el sistema operativo de Microsoft, Windows 11. Durante este contenido, adquirirás los conocimientos esenciales para dominar Windows 11 y aplicarlos a la hora de trabajar en la casa o empresa.

CURSO DE OFIMATICA

El experto en Ofimática es un especialista, cuya formación integral en el manejo de la computadora y las aplicaciones de oficina, le permiten desenvolverse e incrementar su productividad y colaboración en las diversas áreas de la empresa.
